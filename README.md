## WebRTC Load Testing Tool Zakuro について

WebRTC Load Testing Tool Zakuro は libwebrtc を利用した WebRTC SFU Sora 向けの WebRTC 負荷試験ツールです。

## 実験的機能

**現在、すべての機能が実験的機能です。今後、積極的な破壊的変更があります。**
